## Backend Project

### Stack

- Node.js
- Express
- Sequelize
- MySQL2
- dotenv
- cors
- bcrypt
- jsonwebtoken

### Backend Features

- secure registration and login;
- hashed passwords;
- JWT auth;
- full CRUD for posts;
- category filter through query params;
- ownership checks for update and delete;
- smoke tests for auth and CRUD.
